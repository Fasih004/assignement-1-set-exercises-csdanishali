HTML Tags

Below are some commonly used HTML tags with examples:

1. Headings: <h1> to <h6>

Headings are used to define the structure of your content. <h1> is the most important, and <h6> is the least.

<h1>Main Heading</h1>
<h2>Subheading</h2>
<h3>Sub-subheading</h3>

2. Paragraph: <p>

The <p> tag is used to define a paragraph.

<p>This is a paragraph.</p>

3. Anchor: <a>

The <a> tag is used to create hyperlinks.

<a href="https://example.com">Visit Example</a>

4. Image: <img>

The <img> tag is used to embed images.

<img src="image.jpg" alt="Description">

5. Unordered List: <ul> and <li>

The <ul> and <li> tags are used to create bullet-point lists.

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>

6. Ordered List: <ol> and <li>

The <ol> and <li> tags are used to create numbered lists.

<ol>
  <li>First Item</li>
  <li>Second Item</li>
</ol>

7. Division: <div>

The <div> tag is used to group content for styling.

<div class="container">
  <p>This is inside a div.</p>
</div>

8. Span: <span>

The <span> tag is used to style inline content.

<p>This is a <span class="highlight">highlighted</span> word.</p>

9. Table: <table>

The <table> tag is used to create tables.

<table>
  <tr>
    <th>Header 1</th>
    <th>Header 2</th>
  </tr>
  <tr>
    <td>Data 1</td>
    <td>Data 2</td>
  </tr>
</table>

10. Form: <form>

The <form> tag is used to create forms.

<form>
  <label for="name">Name:</label>
  <input type="text" id="name" name="name">
  <button type="submit">Submit</button>
</form>

CSS Properties

Below are some commonly used CSS properties with examples:

1. color

Sets the text color.

p {
  color: #ff6f61;
}

2. background-color

Sets the background color.

body {
  background-color: #f4f4f4;
}

3. font-size

Sets the size of the text.

h1 {
  font-size: 2.5rem;
}

4. margin

Sets the margin around an element.

div {
  margin: 20px;
}

5. padding

Sets the padding inside an element.

div {
  padding: 10px;
}

6. border

Sets the border around an element.

div {
  border: 1px solid #333;
}

7. text-align

Aligns text horizontally.

p {
  text-align: center;
}

8. display

Controls the layout behavior of an element.

div {
  display: flex;
}
